\# Ultimate Image Manager — 终极图片管家



一个为 \*\*Trss-Yunzai (Karin)\*\* QQ 机器人打造的智能图片管理插件，支持\*\*分类存储、去重上传、关键词随机调用\*\*，并提供美观的帮助菜单。



\## ✨ 核心功能



\- 🖼️ \*\*分类管理\*\*：`存入 猫猫` → 自动创建 `images/猫猫/` 目录

\- 🎲 \*\*自然语言指令\*\*：`\&随机猫猫`、`\&随机原神`

\- 📤 \*\*多种存图方式\*\*：

&nbsp; - 回复图片 + `存入 <分类>`

&nbsp; - `偷图`（快速保存引用图）

&nbsp; - `\&添加图片`（URL/本地）

\- 🔒 \*\*权限控制\*\*：管理指令仅限管理员

\- 📊 \*\*全局/分类查看\*\*：`\&图片列表`、`\&随机图片`

\- ❓ \*\*帮助系统\*\*：`\&帮助` → 返回图文指南（支持预存图或文字版）



\## 🚀 安装



1\. 将本文件夹放入 Yunzai 的 `plugins/` 目录

2\. 重启机器人

3\. 首次使用会自动创建 `data/images/` 目录



\## 📝 指令列表



| 指令 | 权限 | 说明 |

|------|------|------|

| `存入 <分类名>` | 管理员 | 回复图片后使用，存入指定分类 |

| `偷图` | 管理员 | 快速保存引用的图片到 `default` |

| `\&随机<关键词>` | 所有用户 | 如 `\&随机猫猫` |

| `\&随机图片 \[分类]` | 所有用户 | 兼容旧格式 |

| `\&图片列表` | 所有用户 | 查看所有图片及编号 |

| `\&删除图片 <编号>` | 管理员 | 删除指定图片 |

| `\&重命名图片 <编号> <新名>` | 管理员 | 重命名 |

| `\&设置图片大小 <数字>MB` | 管理员 | 如 `\&设置图片大小 20MB` |

| `\&帮助` | 所有用户 | 查看使用指南 |



> 💡 分类名支持中文、英文、数字（自动过滤非法字符）



\## 📁 目录说明



\- `resources/help\_guide.png`：可选，放入此图将作为帮助菜单背景

\- `data/images/`：图片自动按分类存储（如 `猫猫/123.jpg`）

\- `data/image\_config.json`：配置文件（最大文件大小等）
